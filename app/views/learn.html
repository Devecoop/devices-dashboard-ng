Lelylan Dashboard

Lelylan Dashboard is a simple, intuitive and realtime solution to control all devices in your house. Try the demo out to see it in action and tell us your opinion about. We'll use your feedback to continuosly improve our solutions.

NOTICE Lelylan Dashboard is released as open source on Github.

Create a Device

When you access the dashboard for the first time you don't have any device. To create your first device click the create device link and fill in the form with the desired fields.

  • Name - Device name
  • Type - Device type defining the device structure (learn more about types)
  • Physical - Physical URI related to your device (leave it empty to simulate the device)
Create device Type selection

When you click select type you are redirected to the type selection page. Here you can pick one type between the popular ones or create a new one. Once you have decided the type for your device, just select it.

Types lists Details link

If you want to check the type definitions simply click the details link. A detailed definition will show up, allowing you to see the type structure and to simulate the device component.

Selection type link

Do I need the physical device URI?

If you want to connect the physical world, you'll need to fill the physical field. Here you can find an introductive article showing you how to create your first connected device. In case you do not have a physical device to connect do not worry, you can leave the field empty and simulate the device behaviour.

Device component

Once you have completed the device definition you can start monitoring and controlling it using the device component. For the curious ones the device component is generated through the new <device> tag.
